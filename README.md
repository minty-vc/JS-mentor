Минималистичный MVP веб-сервиса для проверки гипотезы — **повышение вовлеченности пользователей в онлайн-обучение за счет персонализированной обратной связи**

Проект реализован как быстрый прототип без избыточной архитектуры — с фокусом на продуктовую ценность и скорость

Сценарий: пользователь заходит на сайт, читает короткий учебный модуль, отвечает на 2-3 вопроса, получает обратную связь от ИИ. Без регистрации, БД, без отвлекающего UX

— Чистые **HTML5, CSS3, JS**
— **Groq Cloud API**
